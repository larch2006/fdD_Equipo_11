**KIT Skytree – Sistema Hidropónico Vertical de 7 Niveles**   
Descripción: Es un sistema hidropónico vertical automatizado diseñado para cultivar hasta 35 plantas en espacios reducidos. Incluye sistema de riego automático, temporizador para optimizar el consumo de agua y nutrientes, además de componentes orientados a la agricultura urbana sostenible. Está pensado para hogares y pequeños espacios urbanos. El precio aproximado es 800 soles  
Aporte a MoSH: 

* Sirvió como referencia para implementar un sistema de automatización basado en ESP32, encargado de gestionar los ciclos de riego y el funcionamiento programado del sistema hidropónico.  
* Permitió definir la arquitectura de circulación de la solución nutritiva, considerando el uso de una bomba de agua para la recirculación continua desde el depósito hacia los niveles superiores del sistema.  
* Proporcionó una base para el diseño estructural mediante el uso de tuberías y canastillas de cultivo, favoreciendo una construcción modular, de bajo costo y fácil acceso para el mantenimiento y la sustitución de componentes.  
* Demostró las ventajas de una configuración vertical y compacta, optimizando el aprovechamiento del espacio disponible sin reducir la capacidad de cultivo.  
* Sirvió como referencia para el diseño de un sistema orientado a usuarios con conocimientos básicos de hidroponía, priorizando la facilidad de montaje, operación y mantenimiento.

**NIDO PRO – Sistema Hidropónico Inteligente**   
NIDO PRO es un sistema hidropónico inteligente orientado al cultivo horizontal automatizado. Incorpora sensores y tecnologías de monitoreo para controlar variables como pH, conductividad eléctrica, temperatura y humedad, facilitando la supervisión del cultivo mediante herramientas digitales. Está enfocado en agricultura inteligente y automatización aplicada a la producción sostenible.   
Aporte a MoSH:

* Proporcionó una referencia sobre los sensores necesarios para el monitoreo de variables críticas en un sistema hidropónico, tales como pH, conductividad eléctrica (EC), temperatura, humedad y nivel de agua.  
* Sirvió como base para el diseño de una arquitectura IoT (Internet de las Cosas), permitiendo la recopilación, transmisión y visualización de datos en tiempo real para supervisar el estado del sistema de cultivo.  
* Permitió comprender la importancia de la monitorización remota y la toma de decisiones basada en datos, facilitando la detección temprana de condiciones que puedan afectar el crecimiento de las plantas.  
* Destacó la necesidad de implementar mecanismos para el control y gestión de la solución nutritiva, asegurando que las plantas reciban los nutrientes adecuados para su desarrollo.
